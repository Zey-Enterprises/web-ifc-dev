Industrial {% include glossary.html id="trans-fat" text="trans fat" %} is best minimized. In the United States, partially hydrogenated oils are no longer generally recognized as safe for use in food, but checking labels is still reasonable for some packaged foods.{% include citation.html id="u-s-2018-trans-fat" label="1" %}

{% include glossary.html id="saturated-fat" text="Saturated fat" %} is different. It is not a poison, but high intake is usually not the best default for cardiovascular risk management. Most people do better when saturated fat is moderated and more fat comes from unsaturated sources.{% include citation.html id="american-2021-saturated-fat" label="2" %}

Coconut oil is mostly saturated fat. It can fit if you like it and portions are controlled, but it should not be treated as a special fat-loss food. Count the calories and use it like any other energy-dense ingredient.
