Purpose: - Produce a semantic references section tied to the same `_data/citation.json` source used by `citation.html`. - Keep citation numbering/labels local to each page while preserving canonical source records centrally. Parameters: - `style`: optional citation style. Defaults to `site`. `chicago` currently aliases to the same renderer but is accepted intentionally as a future extension point. - `title`: optional section heading override. Defaults to `References`. - `class`: optional extra root class names.
