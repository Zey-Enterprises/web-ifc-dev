{% comment %} Shared link renderer for glossary and citation output. Purpose: - Keep link target behavior and outbound-icon treatment consistent across the annotation system. - Default glossary-style/internal links to same-tab behavior unless `newtab` is set. - Default citation/external links to new-tab behavior unless `newtab=false` is set. Parameters: - `href`: required URL. - `label`: optional visible label. Defaults to `href`. - `newtab`: optional boolean-like value. When blank, `default_newtab` is used. - `default_newtab`: optional boolean-like fallback for the link type/context. - `class`: optional link class additions.
